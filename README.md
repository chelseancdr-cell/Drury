## ✨ 收合功能

### 操作方式
- **展開/收合**: 點擊面板標題列
- **視覺提示**: 箭頭圖示會旋轉顯示狀態
  - ▼ = 展開
  - ▶ = 收合

### 四大面板

#### 📊 即時統計面板（左上）
**展開時顯示:**
- 監測區域總數
- 黃燈警戒數量
- 綠燈安全數量

**收合時**: 僅顯示標題 "📊 即時統計"

---

#### 🎛️ 控制面板（右上）
**展開時顯示:**
- 月份選擇
- 調查次數選擇
- 縣市選擇（按COUNTYCODE排序）
- 鄉鎮市區選擇（按TOWNCODE排序）
- 顯示模式切換
- 更新地圖按鈕
- 示警標準說明

**收合時**: 僅顯示標題 "🎛️ 控制面板"

---

#### 🎨 圖例說明面板（左下）
**展開時顯示:**
- 綠燈/黃燈顏色對照
- 圓圈大小說明
- 監測頻率資訊
  - 5-7月: 每月2次
  - 8-11月: 每月1次
  - 12-1月: 每周1次

**收合時**: 僅顯示標題 "🎨 圖例說明"

---

#### 📋 因應措施面板（右下）
**展開時顯示:**
- 🟡 黃燈警戒措施
  - 盛花期前45天: 建議施藥
  - 盛花期: 加強巡田防治
- 🟢 綠燈安全措施
  - 田間巡查
- 會議決議依據

**收合時**: 僅顯示標題 "📋 因應措施"

## 💡 使用技巧

### 推薦收合順序

**初次使用時（建議全部展開）:**
1. 先設定好控制面板的查詢條件
2. 查看即時統計了解概況
3. 參考圖例和因應措施

**查詢時（建議收合非必要面板）:**
1. 保持控制面板展開（操作用）
2. 收合其他面板增加地圖可視範圍

**分析時（根據需求展開）:**
1. 需要看統計 → 展開即時統計
2. 需要看措施 → 展開因應措施
3. 需要看圖例 → 展開圖例說明

### 快速操作流程

```
1. 點擊控制面板標題 → 展開
2. 設定查詢條件
3. 點擊「更新地圖」
4. 點擊控制面板標題 → 收合
5. 查看地圖和其他資訊
```

## 🎯 面板狀態記憶

- 面板的展開/收合狀態在頁面重新整理後會重置
- 建議依據使用情境調整面板狀態

## 📱 響應式設計

- 面板寬度固定，確保不重疊
- 內容區域可滾動
- 適合各種螢幕尺寸

## ⌨️ 鍵盤快捷鍵

目前版本暫不支援鍵盤快捷鍵，所有操作都通過點擊完成。
